API Help Page

Products

API Route Description Admin Auth
GET All products /api/product/ Get Products JSON format
(?admin=adminId for admin specific products)
(?new=true for the 5 most recent products)		 No No
GET a single Product api/product/:id Get a single product given the :id No No
POST a product /api/product/:userId Post a product to the database: requires the userId to be a valid adminId Yes Yes
POST a product review api/product/review/post Post a product review to the database; userId and other details in the body of hte request No Yes
GET all product reviews api/product/review/get/:id Get all the reviews for a produtc given the :id No Yes
PUT/Update a product api/product/:userId/:productId Update a product in the database given the :userId, :productId Yes Yes
DELETE a product api/product/delete/:userId/:productId Delete a product given the :userId, product:Id Yes Yes
